ИНФОРМАЦИЯ О ЗАПИСЯХ НА ПРИЁМ К ВРАЧУ
========================================

ГДЕ ОТОБРАЖАЮТСЯ ЗАПИСИ ПОСЛЕ ОТПРАВКИ:
----------------------------------------

1. НА САЙТЕ (для всех пользователей):
   - Страница "Записи на приём" (ссылка в главном меню)
   - URL: /appointments
   - Здесь отображается таблица со всеми записями:
     * ФИО пациента
     * Врач и специальность
     * Дата и время приёма
     * Контактные данные (телефон, email)
     * Статус подтверждения
     * Дата создания записи
     * Комментарии (если есть)

2. В АДМИНИСТРАТИВНОЙ ПАНЕЛИ DJANGO (для администраторов):
   - URL: /admin/app/appointment/
   - Расширенный интерфейс для управления записями:
     * Просмотр всех записей в удобной таблице
     * Фильтрация по врачу, дате, статусу подтверждения
     * Поиск по имени пациента, телефону, email
     * Редактирование записей
     * Подтверждение/отмена подтверждения записей
     * Удаление записей
     * Группировка по датам

3. НА СТРАНИЦЕ КОНТАКТОВ:
   - После успешной отправки формы записи появляется сообщение:
     "Запись успешно создана! Ваша заявка на приём к врачу принята."
   - В сообщении есть ссылка для перехода на страницу "Записи на приём"

4. В БАЗЕ ДАННЫХ:
   - Все записи сохраняются в таблице "Appointments"
   - Доступны через Django ORM для программной обработки

УПРАВЛЕНИЕ ВРАЧАМИ:
-------------------

Врачи управляются через административную панель:
- URL: /admin/app/doctor/
- Можно добавлять, редактировать и удалять врачей
- Можно активировать/деактивировать врачей (неактивные не отображаются в форме)

ОСОБЕННОСТИ:
------------

- Проверка занятости времени: нельзя записаться на уже занятое время к врачу
- Валидация даты: нельзя записаться на прошедшую дату
- Уникальность: один врач не может принимать двух пациентов одновременно
- Статус подтверждения: администратор может подтверждать записи
- Фильтрация: в форме отображаются только активные врачи
