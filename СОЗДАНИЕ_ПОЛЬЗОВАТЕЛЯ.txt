КАК СОЗДАТЬ ПОЛЬЗОВАТЕЛЯ В ПРОЕКТЕ
===================================

СПОСОБ 1: СОЗДАНИЕ АДМИНИСТРАТОРА (СУПЕРПОЛЬЗОВАТЕЛЯ)
-----------------------------------------------------
Этот способ создаёт пользователя с правами администратора, который может:
- Заходить в админ-панель Django (/admin/)
- Добавлять новости через интерфейс сайта
- Управлять всеми данными через админ-панель

ШАГИ:
1. Откройте командную строку (PowerShell или CMD)
2. Перейдите в папку проекта:
   cd "d:\Django VS\DjangoWebProject_Theme6-main"

3. Убедитесь, что база данных создана (выполните миграции, если ещё не делали):
   python manage.py migrate

4. Создайте суперпользователя:
   python manage.py createsuperuser

5. Введите данные:
   - Username (имя пользователя): например, admin
   - Email address: ваш email (можно оставить пустым)
   - Password: придумайте пароль (будет скрыт при вводе)
   - Password (again): повторите пароль

6. Готово! Теперь вы можете войти:
   - На сайте: http://127.0.0.1:8000/login/
   - В админ-панель: http://127.0.0.1:8000/admin/


СПОСОБ 2: РЕГИСТРАЦИЯ ЧЕРЕЗ САЙТ
---------------------------------
Обычные пользователи могут регистрироваться через веб-интерфейс.

ШАГИ:
1. Запустите сервер (если ещё не запущен):
   python manage.py runserver

2. Откройте браузер и перейдите на:
   http://127.0.0.1:8000/registration

3. Заполните форму регистрации:
   - Username (имя пользователя)
   - Password (пароль)
   - Password confirmation (подтверждение пароля)

4. Нажмите "Зарегистрировать"

5. После регистрации вы автоматически попадёте на главную страницу

ВАЖНО:
- Пользователи, зарегистрированные через сайт, НЕ являются администраторами
- Они могут только:
  * Оставлять комментарии к новостям
  * Просматривать контент сайта
- Они НЕ могут:
  * Заходить в админ-панель
  * Добавлять новости через интерфейс сайта


КАК СДЕЛАТЬ ОБЫЧНОГО ПОЛЬЗОВАТЕЛЯ АДМИНИСТРАТОРОМ
---------------------------------------------------
Если вы уже зарегистрировали пользователя через сайт и хотите дать ему права администратора:

1. Откройте админ-панель: http://127.0.0.1:8000/admin/
2. Войдите как суперпользователь
3. Перейдите в раздел "Users" (Пользователи)
4. Найдите нужного пользователя и откройте его
5. Установите галочки:
   - Staff status (доступ к админ-панели)
   - Superuser status (права суперпользователя)
6. Сохраните изменения


ПОЛЕЗНЫЕ КОМАНДЫ
----------------
- Создать суперпользователя: python manage.py createsuperuser
- Показать всех пользователей (через Python shell):
  python manage.py shell
  >>> from django.contrib.auth.models import User
  >>> User.objects.all()
  >>> exit()
