КАК ПРОСМОТРЕТЬ ОТЗЫВЫ (ОБРАТНУЮ СВЯЗЬ)
=========================================

Теперь все отзывы, отправленные через форму обратной связи, сохраняются в базе данных!

СПОСОБ 1: ЧЕРЕЗ АДМИН-ПАНЕЛЬ DJANGO (РЕКОМЕНДУЕТСЯ)
---------------------------------------------------

1. Запустите сервер Django (если не запущен):
   python manage.py runserver

2. Откройте браузер и перейдите по адресу:
   http://127.0.0.1:8000/admin/

3. Войдите с данными суперпользователя:
   - Имя пользователя: admin
   - Пароль: admin123

4. В разделе "APP" найдите "Отзывы" (Feedback)

5. Нажмите на "Отзывы" - вы увидите список всех отправленных отзывов

6. Вы можете:
   - Просматривать все отзывы
   - Фильтровать по дате
   - Искать по имени или email
   - Открывать отдельные отзывы для просмотра деталей
   - Удалять отзывы (если нужно)

В админ-панели отображается:
- Имя отправителя
- Email
- Дата отправки
- Краткая информация об отзыве


СПОСОБ 2: ЧЕРЕЗ БАЗУ ДАННЫХ (ДЛЯ ПРОДВИНУТЫХ)
---------------------------------------------

Отзывы хранятся в таблице "Feedback" в файле базы данных:
   db.sqlite3

Вы можете открыть базу данных с помощью:
- SQLite Browser
- DB Browser for SQLite
- Или через Python shell Django


СТРУКТУРА ОТЗЫВА:
------------------
Каждый отзыв содержит:
- Имя
- Город
- Род занятий
- Пол
- Оценка дизайна сайта
- Оценка актуальности контента
- Согласие на получение новостей
- Разрешение на связь
- Email
- Текст отзыва
- Дата и время отправки


ВАЖНО:
------
- Все новые отзывы автоматически сохраняются в базу данных
- Старые отзывы (отправленные до обновления) не сохранялись
- Отзывы можно просматривать, редактировать и удалять через админ-панель
- Для доступа к админ-панели нужны права суперпользователя
